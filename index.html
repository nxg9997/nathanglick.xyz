About Me

Hi! I'm Nate, a current Game Design & Development student at RIT! I am mostly a programmer, but I like to work on the design for a lot of my projects as well. I am well versed in languages such as: C#, Java, HTML/CSS, and a little bit of Python. As for software, I have lots of experience in Unity3D, Autodesk Maya, Autodesk Inventor, Adobe Photoshop, Adobe After Effects, Vegas Pro, Visual Studio, and Eclipse, among others. I also am studying Japanese, and I am a player for the RIT eSports Heroes of the Storm team!

Detached
2D Platformer
Monogame

About Game
Detached is a 2D platformer where the object of the game is to collect body parts to gain the power to pass through different colored blocks.

Dev Notes
Monogame is a tough "engine" to work in, as the entire game is made purely in C#. This game was my group's final project for GDAPS II.

Pixel Run
2D Platformer
Unity

About Game
Pixel Run is an infinite 2D runner, where the object is to live as long as possible while avoiding blocks that are moving across the screen.

Dev Notes
Pixel Run was made for the MS Paint Game Jam, and only took about 6 hours to make. It was also my first 2D Unity project!

E.C.H.O.
3D Puzzle / Adventure
Unity3D

About Game
E.C.H.O. places you in the shoes of a human test subject who lost their vision during an experiment. Using the power of sound and echolocation, you must travel through a series of dark mazes.

Dev Notes
Echo was made for the Music Game Jam, although the "music" part kinda got away from us during development. Fun fact, the "hard" mode levels are actually puzzle levels that we created for easy mode, but they were so frustrating to play while blind that we almost got rid of them entirely! (You can still play them in hard mode if you wish to suffer ;D)

TimeLock
2D Rogue-Like
GameMaker: Studio

About Game
TimeLock is a 2D rogue-like game, similar to games like The Binding of Isaac or Enter the Gungeon. Each playthrough will have a random floor layout, so it's worth playing multiple times!

Dev Notes
I actually made this back in high school as part of a group project for my game design class. There are probably around 20 different rooms to explore, and I have an extensive password system if you want to have some sick cheat codes. The main mechanic of the game allows you to reverse time, similar to Tracer in Overwatch.
